**Endpoint 1: `POST /scoring/{asset_id}/rescore`** (SCORE-06)
```python
@router.post("/{asset_id}/rescore")
async def rescore_asset(
    asset_id: uuid.UUID,
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
```
- Verify the asset exists and belongs to `current_user.organization_id` (query `CreativeAsset`)
- If not found: raise `HTTPException(status_code=404, detail="Asset not found")`
- Query `CreativeScoreResult` WHERE `creative_asset_id == asset_id`
- If no score record exists: create one with `scoring_status="UNSCORED"`, `organization_id=current_user.organization_id`, `creative_asset_id=asset_id`
- If score record exists: set `scoring_status = "UNSCORED"`, `error_reason = None`, `brainsuite_job_id = None`, `updated_at = datetime.now(timezone.utc)`
- Commit and return `{"status": "queued", "asset_id": str(asset_id)}`

**Endpoint 2: `GET /scoring/status`** (SCORE-08)
```python
@router.get("/status")
async def get_scoring_status(
    asset_ids: str = Query(..., description="Comma-separated asset UUIDs"),
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
```
- Parse `asset_ids` string: split by comma, convert each to `uuid.UUID`, limit to 100 IDs max
- Query `CreativeScoreResult` WHERE `creative_asset_id IN (parsed_ids)` AND `organization_id == current_user.organization_id`
- Return list of dicts: `{"asset_id": str, "scoring_status": str, "total_score": float|None, "total_rating": str|None}`
- For assets with no score record, include them with `scoring_status: "UNSCORED"`, `total_score: None`

**Endpoint 3: `GET /scoring/{asset_id}`** (SCORE-07 detail)
```python
@router.get("/{asset_id}")
async def get_score_detail(

```

- Query `CreativeScoreResult` WHERE `creative_asset_id == asset_id`
