**This session — bugs fixed:**
- `brainsuite_score.py`: Fixed viz file extension detection — 4-level fallback (MIME map → mimetypes → URL path → BrainSuite type hint). Previously `mimetypes.guess_extension("video/mp4")` returned None on most platforms, storing files without extension.
- `object_storage.py`: Fixed `SignatureDoesNotMatch` on presigned URLs — added `_ensure_presigned_client()` using `S3_PUBLIC_URL` as endpoint. HMAC signature must be signed with the browser-reachable hostname, not the Docker-internal one.
- `asset-detail-dialog.component.ts`: Fixed data path — BrainSuite kpis are flat on `legResults[0].kpis` (not on `executiveSummary.categories[i].kpis`). Updated `getSelectedPillarKpiList()` to filter by category name.
- `asset-detail-dialog.component.ts`: Fixed viz toggle buttons not firing on click — added `trackBy: trackVizByKey` to `*ngFor`. Without it, `mousedown` triggers change detection which recreates DOM nodes; the `click` fires on the detached old element.
