    **Component class changes:**

    Add method `onAutoFillToggle(field: any, event: any)`:
    ```typescript
    onAutoFillToggle(field: any, event: any) {
      field.auto_fill_enabled = event.checked;
      if (!event.checked) {
        field.auto_fill_type = null;
      }
      this.saveAutoFillSettings(field);
    }
    ```

    Add method `onAutoFillTypeChange(field: any, event: any)`:
    ```typescript
    onAutoFillTypeChange(field: any, event: any) {
      field.auto_fill_type = event.value;
      this.saveAutoFillSettings(field);
    }
    ```

    Add method `saveAutoFillSettings(field: any)`:
    ```typescript
    saveAutoFillSettings(field: any) {
      this.api.patch(`/assets/metadata/fields/${field.id}`, {
        auto_fill_enabled: field.auto_fill_enabled,
        auto_fill_type: field.auto_fill_type
      }).subscribe({
        next: () => {
          this.snackBar.open('Auto-fill settings saved', '', { duration: 2000 });
        },
        error: () => {
          this.snackBar.open('Failed to save — please try again', '', { duration: 4000 });
        }
      });
    }
    ```

    **Ensure MatSlideToggleModule is in the component's `imports` array.** Check if MatSelectModule is already imported (it likely is for field_type selector) — if not, add it.
